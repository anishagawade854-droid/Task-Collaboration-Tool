# Task-Collaboration-Tool
Board features

Drag and drop tasks between columns (Backlog → In Progress → Review → Done)
Click any task card to open a detailed view with editable fields
Add tasks to any column with the + button or the "New task" button
Delete tasks from cards or the detail panel
Add columns with the + button at the end of the board

Three views

Board — visual kanban with drag-and-drop
List — sortable table with all task details at a glance
Stats — completion rates, status breakdown, and team workload

Per-task details

Title, description, status, priority, assignee, due date, project, and tag — all editable inline in the detail modal
Threaded comments on each task
Overdue dates highlighted in red

Sidebar

Quick project and team filters (clicking opens an AI-powered summary in chat)
Live task counts throughout
